Mailtools Installation Guide


This guide is not meant to be comprehensive, so we recommend that you use many available online resources to become more acquainted with UNIX terminal commands.


Dependency Prerequisites

  1. The following packages need to be installed on the system via apt:
    • python3-pip
    • python3-tk
    • mongodb
    To install, enter the following command in the terminal:
      sudo apt-get install python3-pip mongodb
  2. The following Python3 packages need to be installed on the system via pip3:
    • argcomplete
    • pyyaml
    • pymongo
    • psutil
    • lark-parser
    • pynput
    To install, enter the following command in the terminal:
      pip3 install argcomplete pyyaml pymongo psutil lark-parser pynput

Install

  1. Click to download the latest release of mailtools (i.e. 'mailtools_x.x.x-stable_arch.deb')
  2. Install the package with the following command:  sudo dpkg -i mailtools_x.x.x-Stable_arch.deb
    Expected Results:
        /usr/lib/mailtools directory installed
      /usr/lib/mailtools/bin directory installed
      /usr/lib/mailtools/etc directory installed
      /usr/bin/mailtools excutable link installed
  3. Run mailtools without any arguments. This will create the user's configuration files under $HOME.
    Expected Results:
        /home/$USER/.mailtools directory created
      /home/$USER/.mailtools/config.yml file installed (with user specific settings)
      /home/$USER/.mailtools/results file created (blank file)
      /home/$USER/mail directory created (if it does not exist already)
      /home/$USER/mail/Mailtools link created (points to /home/$USER/.mailtools/results)

Uninstall

    Run the following command:
    sudo dpkg -r mailtools
