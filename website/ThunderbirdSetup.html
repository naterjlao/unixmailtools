Thunderbird Setup Guide


This guide is to integrate mailtools with a Thunderbird Client


Prerequisites

  1. Thunderbird
  2. Mail server running mailtools
  3. An associated account on Thunderbird that points to the mail server

To Setup:

  1. Right-click the mail server account on Thunderbird
  2. Select 'Subscribe...'
  3. Click 'Refresh'
  4. In 'Folder List', select 'Mailtools' and click 'Subscribe'
  5. Click 'OK'

Under the mail server account, there will be a 'Mailtools' folder. This will hold all associated user search results.


To Remove:

  1. Right-click the mail server account on Thunderbird
  2. Select 'Subscribe...'
  3. Click 'Refresh'
  4. Under 'Folder List', select 'Mailtools' and click 'Unsubscribe'
  5. Click 'OK'
