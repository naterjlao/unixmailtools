Frequently Asked Questions

Commonly asked questions about Mailtools

  1. What is Mailtools?

    Mailtools is designed for Debian distribution of Linux. Mailtools is designed with the following features:

    • Smart Search - Mailtools can efficiently search through mail files using user defined queries based on mail content.
    • If-This-Then-That (IFTTT) Rules - Mailtools can execute commands based on user-defined conditions on mail.
    • Scheduled Mail Tasks - Mailtools can schedule automated tasks periodically or at one time as defined by the user.
  2. What mailbox formats are supported?

    As of Mailtools ver 1.3.0, the mbox format is the only mailbox format supported.

  3. Why should user should users choose Mailtools over popular email clients?

    Many companies, especially those dealing with confidential, classified materials such as government agencies and government contractors, only employ private email servers as their central communication system between employees. Using Gmail, which is owned privately and has out-sourced contractors overseas, will prose a huge high security risk to information data. Usually, the type of machines that are employed to manage private mail servers are different flavours of Linux. Debian which is the OS of our virtual machines, is an example of a Linux distribution.

  4. Can you use a different mail client other than Thunderbird?

    Yes. Because of the way that Mailtools is configured, other mail client programs, such as Apple Mail can be used in conjunction with this program. By default, the results of a mail query is stored in /home/username/.mailtools/results file on the email server. Mailtools automatically creates a symbolic link in /home/username/mail called Mailtools that point to this results file. i.e.: /home/username/mail/Mailtools -> /home/username/.mailtools/results Most email clients will see the Mailtools symbolic and it will appear as a "mail folder" in the mail server account. If the mail client checks a different directory, another symbolic can be created at that location that can point to the results file.

  5. Do my emails have to be stored locally?

    Yes. Mailtools only acts on .mbox files local on the same machine.
